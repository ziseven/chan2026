# 台北市立大學 2026 夏季課程計劃

> **Utaipei 2026 Summer Course Plan** | 台北市立大學 2026 夏季課程教學計劃

---

## 📌 專案簡介

本專案是 **台北市立大學 2026 夏季課程（Utaipei 2026 Summer）** 的完整教學規劃網頁，旨在提供學生一個結構化的學習路徑，引導學生深入掌握核心課程主題，並結合實務演練與專題研究以達成最佳學習效果。

---

## 🎯 核心特色

- ✅ **結構化課程**：8週完整的學習進度規劃
- ✅ **理論與實務結合**：核心理論基礎 + 實務案例分析 + 專題研究
- ✅ **團隊協作導向**：強調團隊合作與跨領域學習
- ✅ **美觀響應式網頁**：採用現代化設計，支援各種裝置

---

## 📚 課程內容

### 核心能力培養
- 理論認知深度掌握與跨學科思維
- 實務案例立基規劃與執行能力  
- 團隊協作與跨領域學習交流

### 每週主題概覽
| 週次 | 主題 |
|------|------|
| Week 01 | 課程導入與環境建置 |
| Week 02 | 核心理論基礎（一） |
| Week 03 | 核心理論基礎（二） |
| Week 04 | 實務案例與分析 |
| Week 05 | 期中成績發表 |
| Week 06 | 進階技術與應用 |
| Week 07 | 專題深化與協作 |
| Week 08 | 期末成績發表 |

## 🚀 快速開始

### 線上預覽
- [GitHub Pages](https://oceanicdayi.github.io/Utaipei_2026_summer/) （原始倉庫）
- 或直接打開本專案的 `index.html` 檔案

### 本地運行
```bash
# 克隆專案
git clone https://github.com/ziseven/chan2026.git

# 打開 index.html
open index.html  # macOS
start index.html # Windows
xdg-open index.html # Linux
```
